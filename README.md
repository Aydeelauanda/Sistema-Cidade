
 Sistema-Cidade

Projeto de modelagem urbana desenvolvido para a disciplina de Programação, focado em POO (Programação Orientada a Objetos) e UML.

 Documentação (Diagramas)
Os diagramas de Classe, Pacotes, Sequência e Componentes podem ser acessados aqui:

 Link dos Diagramas:** [C:\Users\ferna\documents\projeto_cidade_finalizado.asta]

 Arquitetura do Código
O projeto está dividido em três pacotes principais:
1. **Model:** Contém a classe abstrata `ModuloUrbano` e a interface `IConectavel`.
2. **Infra:** Implementa `Estrada` e `Ponte`.
3. **Edificacoes:** Implementa `Casa` e `Hospital`.

 Tecnologias Utilizadas
- Java 17
- UML (StarUML/Astah)
- GitHub para versionamento
